# Web3 Wallet Connector

A clean, production-ready Ethereum wallet connection interface built with React and TypeScript.

## Features

- Connect to MetaMask and other Ethereum wallets
- Display wallet address with copy functionality
- Show real-time ETH balance
- Detect network changes automatically
- Handle account switching gracefully
- User-friendly error handling
- Clean, responsive UI

## Tech Stack

- React 18 + TypeScript
- Ethers.js v6
- Tailwind CSS for styling
- Vite for build tooling

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Usage

The wallet connector automatically handles:
- Connection requests and user rejection
- Account and network change events
- Balance updates
- Error states and user feedback

Simply import the `WalletConnector` component and it manages all wallet state automatically.
